CHAPTER 1
INTRODUCTION

1.1 Background

CARMANDU is a modern web platform designed to showcase luxury cars and provide users with detailed information and appointment booking options. The website features a curated selection of high-end vehicles, including brands such as Ferrari, Lamborghini, and more. CARMANDU aims to offer car enthusiasts and potential buyers an engaging experience, allowing them to explore car specifications, view images, and easily schedule showroom visits. The platform supports digital transformation in the automotive sector by making luxury car information and booking accessible online.

1.2 Scope

The scope of this project is to develop a fully responsive frontend website for CARMANDU, focusing on user interface design and interactive client-side functionality. The project includes the creation of multiple car detail pages, a booking dialog, and responsive navigation. Backend features such as database integration, payment processing, and server-side operations are not included in this frontend-only implementation.

1.3 Project Description

This project involves building a visually appealing and responsive car showcase website from scratch. Key components include a homepage with a grid layout displaying various luxury cars, individual car detail pages with images and specifications, an appointment booking system via modal dialog, and navigation to About, Contact, and Login pages. The website supports dark mode and mobile-friendly navigation. Technologies used are HTML5 for semantic structure, CSS3 for responsive styling with media queries, and vanilla JavaScript for interactive features.

1.4 Objectives

  • Create a responsive car showcase website using HTML, CSS, and JavaScript.
  • Design a user-friendly interface compatible with mobile, tablet, and desktop devices.
  • Build interactive car detail pages with appointment booking functionality.
  • Implement modern web features such as hamburger menu and dark mode toggle.
  • Develop skills in frontend web development and responsive design techniques.
