1. Introduction

This report documents the development of CARMANDU, a car showcase website designed to provide users with information about luxury cars and facilitate appointment bookings for visits. The website features a modern, responsive design and interactive elements to enhance user experience. The project was completed as part of the Minor Project requirement for the academic session 2024-2025.

2. Objectives

  • To design a visually appealing and responsive car showcase website.
  • To implement individual pages for each car with detailed information.
  • To provide a booking system for users to schedule appointments.
  • To ensure ease of navigation and accessibility across devices.

3. Methodology

The website was developed using HTML and CSS for structure and styling. JavaScript was used to add interactivity, such as the appointment booking dialog. The design process included wireframing, prototyping, and iterative development based on user feedback. Responsive design principles were applied to ensure compatibility with desktops, tablets, and mobile devices.

  1. Requirement analysis and planning.
  2. Designing wireframes and layout.
  3. Developing core pages: Home, Car Details, About, Contact, Login.
  4. Implementing appointment booking functionality.
  5. Testing for usability and responsiveness.

4. Website Features

  • Home Page: Displays a grid of luxury cars (Ferrari, Lamborghini, etc.) with links to individual detail pages.
  • Car Detail Pages: Each car has a dedicated page with images, price, description, and a "Book an Appointment" button.
  • Appointment Booking: Clicking the button opens a dialog box with editable visit information.
  • Responsive Design: Layout adapts for mobile and tablet screens.
  • Navigation: Easy access to About, Contact, and Login pages.
  • Dark Mode: Toggle for light/dark theme for better accessibility.

5. Results & Discussion

The CARMANDU website was successfully developed and deployed. User testing indicated high satisfaction with the interface and booking process. The grid layout and responsive design ensured usability across devices. The appointment dialog provided a simple way for users to schedule visits. Challenges included optimizing the grid for mobile screens and ensuring accessibility, which were addressed through CSS media queries and semantic HTML.

6. Conclusion

The project met its objectives by delivering a functional and attractive car showcase website. The experience improved my skills in web design, responsive layout, and user interaction. Future enhancements may include integrating a backend for appointment management and expanding the car database.
