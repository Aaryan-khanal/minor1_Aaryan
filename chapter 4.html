4.2 Debugging

Several issues were encountered during development, including layout discrepancies and JavaScript errors. These challenges were addressed through iterative testing and debugging processes. Key issues resolved included:

  • Navigation Menu Responsiveness: Fixed hamburger menu toggle functionality for mobile devices by implementing proper event listeners in script.js
  • Form Validation Logic: Enhanced email domain validation in login system to ensure only @carmandu.com addresses are accepted
  • Shopping Cart Calculations: Resolved price calculation errors to ensure accurate total updates when automotive items are added or removed
  • CSS Layout Issues: Adjusted responsive breakpoints and grid layouts to improve car brand display on smaller screen sizes
  • Image Loading Problems: Standardized image paths and verified all car brand logos and product images load correctly
  • Car Brand Page Links: Fixed navigation issues between individual car brand pages (audi.html, bmw.html, ferrari.html, etc.)
  • Dark Mode Implementation: Resolved CSS conflicts to ensure dark mode applies consistently across all car brand pages

This iterative approach not only improved the overall stability and performance of the website but also enhanced the user experience by eliminating potential obstacles and ensuring a seamless interaction with the automotive showcase site.

4.3 Results

The final product is a functional Carmandu website that successfully showcases various automotive products and car brands. The visual design closely mirrors professional automotive e-commerce standards while maintaining consistency in layout, colour schemes, and typography that reflects modern automotive excellence. This achievement reflects the successful implementation of HTML, CSS, and JavaScript to create key functionalities and visual elements, providing users with an intuitive and engaging interface when accessing the Carmandu automotive website.

Key accomplishments include:

  • Multi-page Navigation: Seamless navigation between homepage (index.html), individual car brand pages (audi.html, bmw.html, ferrari.html, tesla.html, mercedes.html, etc.), and login page
  • Automotive Showcase: Comprehensive display of car brands including Audi, BMW, Ferrari, Lamborghini, Mercedes, Porsche, Tesla, Bugatti, and Rolls Royce with detailed information
  • Interactive Shopping Cart: Fully functional cart system with add-to-cart, quantity management, and real-time price calculations for automotive products
  • User Authentication: Secure login system with domain-specific email validation (@carmandu.com) and password verification
  • Responsive Design: Mobile-friendly layout with hamburger menu implementation for smaller screens across all car brand pages
  • Dark Mode Feature: Complete theme switching capability with user preference storage across the entire automotive website
  • Automotive Branding: Professional representation of the automotive industry through visual design and car brand presentation
  • Search and Filter: Dynamic search functionality for finding specific car brands and automotive products
  • Contact Integration: Multiple communication channels for customer inquiries and automotive consultation

The website successfully demonstrates practical web development skills while serving as an effective digital platform that showcases and promotes various automotive brands and products, creating an engaging user experience for car enthusiasts and potential buyers.
